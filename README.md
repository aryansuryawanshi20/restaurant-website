# Responsive WordPress One-Page Website

## Project Description
This project is a single-page WordPress website designed and developed to practice real-world WordPress development skills. It demonstrates the use of a page builder along with custom front-end code to create a structured, responsive, and visually clean website.

## Tech Stack
- WordPress
- HTML
- CSS
- JavaScript (basic usage)
- Elementor Page Builder
- Custom styling and layout adjustments

## Features Implemented
- Functional navigation bar with section-based scrolling
- Mobile-friendly and responsive layout
- Custom-designed sections including About, Gallery, and Contact
- Image handling and content organization
- Simple and optimized front-end structure

## What I Learned
- Working with WordPress using page builders and custom code
- Creating layouts using HTML and CSS inside WordPress
- Understanding WordPress structure and local development
- Exporting WordPress sites and deploying them live

## Hosting & Deployment
The project was built in a local WordPress environment and later converted into a static website for deployment on a live hosting service.
